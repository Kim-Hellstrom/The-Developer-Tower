# The-Developer-Tower 🗼
Features Developer projects ranging between software, hardware and creative ideas.

List of items:
* What the project does
* Why the project is useful
* How users can get started with the project
* Where users can get help with your project
* Who maintains and contributes to the project

## What the Project does:
The Developer tower consists of multiple layers of various projects.
Each layer exists with a single major theme, which then gets divided into smaller projects
which holds smaller tasks.

---
Think of it as:

Theme = Floor/Layer

Project = Corridors

Tasks = Rooms

## Why the project is useful
By using a tower structure, we can achieve a clean environment with several different
projects without it becoming clustered or confusing when navigating the space.

## How users can get started with the project
The repo isn't static, it's dynamic. Meaning projects will change, floors will be added.
If you can't find an already made floor to suit your project or task, then you may add
a new floor to the task. Just remember to document what the floor is about and what is expected of
people to add to the new floor.

## Where users can get help with the project
This is a very open structured project. Of course you can reach out to me. I have my email set up, and i check
fairly regularly. Other great sources includes github documentation as well as reddit or Youtube, whom might have
already solved your issue. I'm always open to collaborate together, so if you need a dedicated study partner as you
develop your or others project or need someone to help with accountability. I'm happy to assist!

## Who maintains and contributes to the project
Currently it's only the owner that has contributed. Feel free to also contribute or design a small project. But please stick
to your project until finished, keep it simple. I would love to see everyone's ideas.
